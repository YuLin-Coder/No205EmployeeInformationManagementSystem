## 项目介绍
基于java+swing+awt的员工信息管理系统：集成员工信息的录入、修改、查找、删除等功能于一体的系统。

## 功能介绍

- 基本功能：登录，注册
- 员工信息：员工信息的录入、修改、删除、按工号进行查找
